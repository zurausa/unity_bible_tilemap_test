===========================================
「Unity ゲーム プログラミング・バイブル」サンプルプログラム

No.10 TilemapDungeon
===========================================

これは、株式会社ボーンデジタル社 発行・発売の「Unity ゲーム プログラミング・バイブル」のTilemapの章のサンプルプロジェクトです。

■注意
このプロジェクト、サンプルコードはTilemapのデモ用のものです。実際のゲームを想定したコード・構成にはなっていないので注意してください。

■ディレクトリ構成
TilemapDungeonディレクトリは、読者のあなたが実際に作業するディレクトリです。
TilemapDungeonCompleteディレクトリは、完成後のシーンや、作業後のアセットがあるディレクトリです。
Tilemapディレクトリは、Unity Technologies社の2d-extrasです。リビジョンは、「2496f0e58a000272faa5af8ae07ef1fea9cd7dc5」のものを利用しています。

■ソースコード解説
Coin.cs
　コインをつかさどるクラスです。Collectメソッドを呼ばれた際、事前にセットされたコールバックを呼び出します。

Player.cs
　プレイヤーをつかさどるクラスです。キーボード入力でのプレイヤーの移動を管理します。コインに接触したタイミングで、CoinクラスのCollectメソッドを呼び出します。

DemoSceneManager.cs
　デモのシーンを管理するクラスです。ゲームのプレイ前、プレイ中、プレイ後の状態管理を行います。また、コインにコールバックの設定を行います。プレイヤーがコインを獲得した際の処理をこのクラスが担います。
